This Hugging Face Space showcases **WhiteRabbitNeo Llama 3 8B v2.0**, a cybersecurity-focused large language model built on Meta's Llama 3 8B architecture. It is designed to assist with security research, code analysis, and offensive/defensive security tasks.

**Getting Started**

To interact with this Space, enter a prompt in the text area and click **Run Model**. You can also upload a `.txt` or `.pdf` file to use as context for the model.

**What it does**

WhiteRabbitNeo is fine-tuned on cybersecurity datasets to provide expert-level assistance with topics such as vulnerability analysis, penetration testing methodology, secure code review, and threat modelling. It accepts free-text prompts and optional file uploads, and generates responses using configurable temperature and token-length settings.
